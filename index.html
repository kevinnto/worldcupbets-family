Lokalt läge: dela med gänget genom att committa filen
🔴 Lokalt läge · Koppla upp live på ~10 minuter
  1. Skapa ett Firebase-projekt

    Gå till console.firebase.google.com · klicka Add project · välj ett namn · klicka dig igenom stegen · Create project.

  2. Aktivera Realtime Database

    I vänstermenyn: Build → Realtime Database → Create database. Välj närmaste region (t.ex. europe-west1). Välj Start in locked mode · klicka Enable.

  3. Uppdatera säkerhetsreglerna

    Klicka på fliken Rules · radera allt · klistra in nedanstående · klicka Publish.

    {
  "rules": {
    "balances": {
      ".read": true,
      ".write": true,
      "$date": { "$player": {
        ".validate": "newData.isNumber() && newData.val() >= 0"
      } }
    },
    "bets": {
      ".read": true,
      ".write": true,
      "$date": { "$player": {
        ".validate": "newData.isString() && newData.val().length <= 200"
      } }
    }
  }
}
  4. Hämta din konfiguration

    Klicka på kugghjulet (⚙) → Project settings → rulla ned till Your apps → klicka webb-ikonen </> → registrera appen med valfritt namn → kopiera det firebaseConfig-objekt som visas.

  5. Klistra in och committa

    Öppna js/config.js i repot → ersätt hela window.FIREBASE_CONFIG = {…} med din kopierade config → committa och pusha. Kom tillbaka hit och ladda om — statusindikatorn ovan ska bli 🟢 Live.

    ⚠️ Kom ihåg att sätta Workflow permissions → Read and write under Settings → Actions i repot, annars kan GitHub Actions inte spara resultatdata.
